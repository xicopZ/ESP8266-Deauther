# ESP8266-Deauther

## Parts You Will Need:
1) NODEMCU ESP8266 Board
2) Jumper Wires
3) Breadboard
4) 0.96' OLED Display
5) USB-C Data Cable
6) PC
   
## Full Tutorial
1) Create the circuit as shown on the schematic. Here is a text representation of the connections:
### Display
SDA + D1
SCL + D2
GND + GND
VCC + 3V3
### Switches
OK + CLK
DOWN + D5
UP + D3
